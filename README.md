# Actividad_1_Progra_IV

## Investigacion de los 5 puestos:
### Oferta 1:

Nombre del puesto:

Ingeniero de software


Tecnologías requeridas:

TypeScript
Git


Habilidades técnicas:

Experiencia construyendo UI responsivas con las mejores practicas.

Conocimiento de control de versiones usando git.


Habilidades blandas:

Estar dispuesto a colaborar con otras personas.


Responsabilidades principales:

Colaborar con otras personas para el correcto desarrollo de los sistemas.

Ingeniero de software

## Patrones comunes:

Solo una de las ofertas pedía de manera explicita el saber ingles; sin embargo, todas las ofertas laborales estaban en ingles por lo que se sobreentiende que se debe tener un buen nivel de ingles para todos los casos.

Todos piden estar dispuesto a colaborar con otras personas y habilidades de comunicación para el correcto trabajo en equipo.

También se pide bastante la capacidad de resolver problemas.

## Perfil profesional objetivo:

Alguien que sea capaz de resolver problemas.
Hablar ingles.
Tener un amplio conocimiento en diferentes lenguajes de programación.
Ser bueno trabajando en equipo.

## Justificación:

Escogí este perfil porque es lo que se suele buscar en las ofertas de trabajo, siempre es indispensable tener un buen nivel de ingles y si se tiene algún tipo de título ubersitario que lo corrobore es mucho mejor pues esto da más credibilidad y asegura un mejor salaio, también ser bueno trabajando en equipo pues son muy pocos los puestos donde alguien trabaja totalmente solo entonces es muy importante el trabajo en equipo ademas, en todas las ofertas laborales investigadas se pide trabajar en equipo, conocimiento en diferentes lenguajes de programación para poder cubrir las diferentes necesidades de las empresas y la capacidad de resolver problemas para ser capaz de enfrentarse a las diferentes situaciones que se presenten en la vida laboral.
